fn to_univariate_at_point(
    &self,
    at: usize,
    point: &[F]
) -> Option<SparsePolynomial<F>>

Given an index of a variable i, and a point at in $F^n$ reduce the multivariate polynomial to a single variate polynomial $s(x_i)$ by partially evaluating g at point $(a_1,…,a_{i-1},X_i,a_{i+1},…,a_n)$.

Return None if the at position was wrong or point dimentions did not match the expected ones.
